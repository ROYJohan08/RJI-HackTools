# OSINT
## Holehe
Holehe permet de rechercher les comptes crée avec un email donné.
### Installation
```
git clone https://github.com/megadose/holehe.git
cd holehe/
python3 setup.py install
```
### Usage
```
holehe test@gmail.com
```
